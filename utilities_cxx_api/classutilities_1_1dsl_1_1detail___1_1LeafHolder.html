Detailed Description

template<typename T>
class utilities::dsl::detail_::LeafHolder< T >

API for interacting with the leaf when its type is known.

Template Parameters
TThe possibly cv-qualified type of the object being held.

This layer knows the type of the leaf, including cv-qualifiers. It does not however, know whether we aliased or copied the original object.

Type of a (possibly) mutable pointer to an object of value_type.

If !is_const then this will be a mutable pointer to an object of value_type. If is_const then the pointer will be read-only. Thus this typedef ensures const-consistness with T.
