Detailed Description

template<typename T, typename = void>
struct utilities::type_traits::IsPrintable< T, typename >

Used to determine if an object can be forwarded to an std::ostream.

This is the primary template for checking if an object of a given type can be printed out by passing it to an std::ostream. The class IsPrintable will contain a constexpr static bool member value whose value is true if the T can be printed out and false otherwise. The primary template catches all the cases where T can not be printed out.
