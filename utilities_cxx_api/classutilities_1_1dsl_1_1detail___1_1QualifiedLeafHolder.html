Detailed Description

template<typename T>
class utilities::dsl::detail_::QualifiedLeafHolder< T >

Leaf holder accounting for aliasing vs. copying.

Template Parameters
TThe cv- and reference-qualified type of the object.

The DSL needs to ensure that temporary objects remain in scope until the DSL is fully evaluated. This requires tracking the type of the original leaf object, which is the purpose of this layer of the holder.
